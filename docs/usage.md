# STViewer Usage

For the full tutorial, please read:

- `docs/tutorial.md`

## Quick Start

```bash
stviewer
```

Or:

```bash
bash scripts/start.sh
```

Then open:

- [http://localhost:8501](http://localhost:8501)

## What You Need

- a CSV file
- required columns: `x`, `y`
- one column to use as `cellid`

## What You Can Export

- `selected_cells.csv`
- `annotated_regions.csv`
